vim-利用Vundle管理插件

Posted on 2017-12-17 In Vim Disqus:

Vundle是Vim bundle的一个简称，一个Vim插件管理器。开源项目地址位于Vundle.vim.git
只需要在.vimrc文件内加上几行，你就可以使用它来下载、卸载、管理各种位于github、官方插件仓库等地方的插件。

安装

  1. 安装需要git的支持，采用克隆的方式先克隆到本地。

    git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

  2. 在配置文件内加上几行。

  3. 运行:PluginInstall命令就会将指定的插件进行安装。
