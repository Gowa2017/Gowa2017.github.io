Vim常用插件清单

发表于 2019-01-03 | 分类于 Vim |

个人所有，最近需要用 js 来干一些活，但是用 ide 感觉太重量级了。webStorm 或者 vs code 也好，感觉启动起来都慢慢的，直接用 vim 怕不是要好点哦。所以把我常用的配置都整理了一下。

常规插件

  • Vundle 插件管理。
  • molokai主题
  • ctrlp.vim 路径模糊搜索
  • NERDTree 文件夹浏览器
  • Vim Markdown Vim MarkDown 支持
  • AndrewRadev/splitjoin.vim 在单行语句和多行语句间变换
  • UltiSnips 代码片段插件
  • vim-airline 状态栏加强
  • tagbar 文件结构，标签浏览
  • vim-easygrep 多个文件间查找和替换
  • tabular 文本过滤与对齐
  • vim-syntastic/syntastic 语法检查
  • YouCompleteMe 用来进行补全的，效果不错，非常强大

Go

  • fatih/vim-go Go语言的Vim 支持插件

    JavaScript

  • pangloss/vim-javascript JavaScript 缩进与语法支持

  • maksimr/vim-jsbeautify 格式化 js/json/html/css/jsx 等
  • tern_for_vim 补全 JS 用

关于如何启用，使用 Vundle 作为插件管理，然后添加 YouCompleteMe 和 tern_for_vim ，然后 :PluginInstall。
