关于动态推流的设置(nginx+RTMP)

基本原理是 nginx 开启 rtmp 的支持，当访问 rtmp 链接的时候，执行外挂进行，进行推流命令，就行流的发布。

也就是说在 nginx 的配置中配置一下外挂 shell 脚本，在 shell 脚本中进行 ffmepg 的推流操作。
