项目地址

GNU Parallel

GNU parallel 是一个用来在单个或多个计算机上并行执行任务的 shell 工具。一个任务可以一单个的命令或者一个小小的脚本，他们都必须在输入的每一行上进行执行。典型的输入是 一个文件列表，一个主机的列表，用户的列表，等等。一个任务也可以是一个从管道读取输入的命令。 GNU Parallel 可以将输入进行分块然后并行的通过管道传递给命令。
